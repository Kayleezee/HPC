II. Compilation
---------------
In order to compile, use the Makefile:

For barrier_synch
  $ make barrier_synch

For the matrix multiply
  $ make matrixmultiply
  
For the latex documentation
  $ make approximatepi

To delete the compiled files:
  $ make clean

III. Usage
----------
Matrix Multiplication (The time measurement results will be printed to the standard output):
  $ matvecmult
